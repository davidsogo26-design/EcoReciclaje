# EcoReciclaje

EcoReciclaje is a mobile application developed as an academic project to improve waste management processes within an educational institution.

The application was built using Kotlin in Android Studio and uses Firebase Firestore as its cloud database solution.

## Project Objective

The objective of EcoReciclaje is to help users manage recycling and waste disposal activities more efficiently by providing a centralized platform for information management.

## Technologies Used

* Kotlin
* Android Studio
* Firebase Firestore
* Android SDK

## Main Features

* Information registration and management
* Cloud database integration
* Data storage and retrieval
* User-friendly mobile interface
